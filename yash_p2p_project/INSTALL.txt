MINIMUM STEPS TO RUN IN SAP (5 minutes):
--------------------------------------------------------------------------------

1. UPLOAD PROGRAM 1 (P2P Dashboard):
   - Open SE38
   - Program name: YASH_P2P_DASHBOARD → Enter → Create
   - Attributes: Type = Executable, Status = Test Program → Save
   - Open file: abap/YASH_P2P_DASHBOARD.abap
   - Paste all code → Ctrl+S (Save) → Ctrl+F3 (Activate)

2. UPLOAD PROGRAM 2 (Inventory Report):
   - Open SE38
   - Program name: ZMM_MATERIAL_INVENTORY_REPORT → Create
   - Paste: abap/ZMM_MATERIAL_INVENTORY_REPORT.abap
   - Save → Activate

3. CREATE MESSAGE CLASSES (SE91):
   - Class: YASH_MSG → Message 001 Type I Text: &1 → Save
   - Class: ZMM_MSG  → Message 001 Type I Text: &1 → Save

4. CREATE T-CODES (SE93):
   - ZP2P_DASH → Program: YASH_P2P_DASHBOARD → Save
   - ZMM_INV   → Program: ZMM_MATERIAL_INVENTORY_REPORT → Save

5. RUN:
   - /nZP2P_DASH → Enter Company Code YM01 → F8 (Execute)
   - /nZMM_INV   → Enter Plant PL01 → F8 (Execute)
